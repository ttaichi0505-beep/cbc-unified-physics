# Complex Binary Cosmology (CBC)

## 4つの力を統一する、たった1行のルール

> **「各ピクセルの状態は、近傍のベクトル和の方向に遷移する」**

### シミュレーション結果

| 力 | 結果 |
|---|------|
| 強い力 | 陽子100%安定 / 中性子0%安定 |
| 弱い力 | 左バイアス1.000（パリティ破れ） |
| 重力 | 変位7.00ピクセル |
| 電磁気力 | 同じ電荷:反発 / 異なる電荷:引力 |
